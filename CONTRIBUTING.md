# 🤝 Contributing to AI Code Review Platform

Thank you for your interest in contributing!

## 🌟 Ways to Contribute

- 🐛 Report bugs
- 💡 Suggest features
- 📝 Improve documentation
- 🔧 Submit code changes
- 🧪 Add tests

## 🚀 Getting Started

### 1. Fork the Repository
```bash
git clone https://github.com/sanyaprem/ai-code-review-platform.git
cd ai-code-review-platform
```

### 2. Set Up Development Environment
```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Create a Branch
```bash
git checkout -b feature/your-feature-name
```

## 📋 Development Guidelines

### Code Style

We follow PEP 8 and use automated tools.

### Writing Good Commits

Follow Conventional Commits:
- `feat:` New features
- `fix:` Bug fixes
- `docs:` Documentation
- `test:` Tests
- `refactor:` Code refactoring

## 📤 Submitting Changes

1. Update your branch
2. Run tests
3. Push changes
4. Create Pull Request

## 🐛 Reporting Bugs

Please include:
- Bug description
- Steps to reproduce
- Expected vs actual behavior
- Environment details

## 💡 Feature Requests

Please describe:
- Feature description
- Use case
- Proposed solution
